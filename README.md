## Description

Email Services with [Nest](https://github.com/nestjs/nest) framework TypeScript repository.

## Installation

```bash
$ npm install
```

## Running the app in Offline mode

```bash
# development
$ npm run offline
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
